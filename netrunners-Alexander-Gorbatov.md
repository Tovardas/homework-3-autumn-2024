# Настройки 
### Общие
https://ads.vk.com/hq/settings

- Настройки. На странице настроек отображаются: вкладки "Общие", "Уведомления", "Права доступа", "История изменений"

- Настройки. Общие. При изменении поля "ФИО" и нажатии кнопки сохранить изменение сохраняется и отображается новое ФИО 
- Настройки. Общие. При изменении поля "ИНН" и нажатии кнопки сохранить изменение сохраняется и отображается новый ИНН
- Настройки. Общие. На странице общих настроек отображаются: аватар пользователя, ID кабинета, тип кабинета, формы контакты: 
поля "Телефон" и "Email", кнопка "Добавить Email", реквизиты: поля "ФИО" и "ИНН", интерфейс: поле "Название кабинета",
Accordeon "Язык интерфейса", связанные кабинеты: кнопка "Привязать кабинет myTarget", доступ к API: ссылка "Подробнее о доступе",
кнопка "Запросить доступ к API"; кнопки "Выйти из других устройств", "Удалить кабинет"
- Настройки. Общие. Если внести изменения в любое из полей (Например, "Телефон"), появляются кнопки "Сохранить" и "Отменить"
- Настройки. Общие. Если нажать на кнопку "Отменить" (например, изменили значение в поле "Телефон" на "+79778994111",
а до этого было "+79778994112") все изменения отменяются (в поле "Телефон" стоит значение до изменений - "+79778994112")
- Настройки. Общие. Если нажать на кнопку "Сохранить" (например, изменили значение в поле "Телефон" на "+79778994111",
а до этого было "+79778994112") все изменения сохраняются (в поле "Телефон" стоит значение - "+79778994111")
- Настройки. Общие. Ошибка "Некорректный номер телефона", если номер в поле "Телефон" не соответствует виду "+" и 11-13 цифр
- Настройки. Общие. Невозможно ввести более 14 символов в поле "Телефон"
- Настройки. Общие. Если нажать на кнопку "Добавить email" появляется поле для ввода еще одного адреса электронной почты
- Настройки. Общие. Если нажать на кнопку крестик рядом с полем для ввода дополнительного email, поле для ввода исчезает
- Настройки. Общие. Ошибка "Некорректный email адрес" при вводе email, не удовлетворяющего условию email = имя + @ + домен
- Настройки. Общие. Ошибка "Обязательное поле" при пустом поле Email
- Настройки. Общие. Ошибка "Обязательное поле" при пустом поле "ФИО"
- Настройки. Общие. Ошибка "Обязательное поле" при пустом поле "ИНН"
- Настройки. Общие. Ошибка "Минимальная длина 12" при вводе ИНН короче 12 символов
- Настройки. Общие. Невозможно ввести более 12 символов в поле "ИНН"
- Настройки. Общие. Ошибка "Некорректный ИНН" при вводе в поле "ИНН" не цифры
- Настройки. Общие. Поле "Язык интерфейса". При нажатии на кнопку "RU" в поле "Язык интерфейса" язык интерфейса меняется на русский
- Настройки. Общие. Поле "Язык интерфейса". При нажатии на кнопку "EN" в поле "Язык интерфейса" язык интерфейса меняется на английский
- Настройки. Общие. При нажатии на кнопку "Выйти из других устройств" появляется сообщение "Активные сеансы на других устройствах успешно завершены."
- Настройки. Общие. При нажатии на кнопку "Удалить кабинет" появляется окно для подтверждения удаления кабинета

### Уведомления
https://ads.vk.com/hq/settings/notifications

- Настройки. Уведомления. На странице настроек уведомлений отображаются: формы "Способы получения": переключатель email, кнопка 
"Подключить" (Сообщение в Telegram), "Основные": Checkbox("Финансы", "Модерация", "Рекламные кампании" "Правила для объявлений", "Изменения в API"),
"Новости и акции": Checkbox("Новости", "Мероприятия", "Акции, спецпредложения и прочие")
- Настройки. Уведомления. Если внести изменения в любое из полей (например, в поле "Финансы"), появляются кнопки "Сохранить" и "Отменить"
- Настройки. Уведомления. Если нажать на кнопку "Отменить" (например, деактивировали чекбокс "Финансы") все изменения 
отменяются (чекбокс "Финансы" активен)
- Настройки. Уведомления. Если нажать на кнопку "Сохранить" (например, деактивировали чекбокс "Финансы") все изменения 
сохраняются (чекбокс "Финансы" не активен)

### История изменений
- Настройки. История изменений. На странице отображается список изменений, соответствующий параметрам поиска, активным фильтрам и выбранному диапазону дат.
- Настройки. История изменений. При нажатии на кнопку "Фильтр", появлятеся модальное окно
  ![image](https://github.com/user-attachments/assets/87c8607b-dd5d-4ff4-9107-22a9cbff8414)

  * Фильтр. При выборе "Тип объекта", "Что изменилось", "Автор изменения", менются параметры выборки.
  * Фильтр. При нажатии на кнопку "Выбрать все", активируются все доступные параметры выборки. Сама кнопка меняется на "Сбросить".
  * Фильтр. При нажатии на кнопку "Сбросить", все активные параметры выборки перестают быть активными. Сама кнопка меняется на "Выбрать все".
  * Фильтр. При нажатии на параметр поиска, чекбокс с соответсвющим параметром выборки изменит свое состояние на противоположное (Активен/Неактивен). Состояние чекбокса сохраняется при переключении между "Тип объекта", "Что изменилось", "Автор изменения".
  * Фильтр. При введении запроса в графу "Поиск", на странице остаются параметры выборки, которые содержат в себе текст запроса.
  * Фильтр. При нажатии кнопки "Сбросить все", все параметры выборки во всех вкладках ("Тип объекта", "Что изменилось", "Автор изменения") меняют свое состояние на неактивное.
  * Фильтр. При нажатии на кнопку "Отмена", модальное окно закрывается без сохранения изменений.
  * Фильтр. При нажатии на область вне модального окна, модальное окно закрывается без сохранения изменений.
  * Фильтр. При нажатии на кнопку "Сохранить", параметры выборки добавляются в активные фильтры, а модальное окно закрывается.
  * Фильтр. Активыне фильтры отображаются перед списком изменений и сгруппированы по "Тип объекта", "Что изменилось" и "Автор изменений"
    
    ![image](https://github.com/user-attachments/assets/32664bb3-96a7-455c-98f0-cbbfd9ac2245)
    
  * Фильтр. При нажатии на группу активных фильтров, появляется список доступных в этой группе параметров выборки. При нажатии кнопки "Сохранить", сохраняются новые параметры выборки.
    
    ![image](https://github.com/user-attachments/assets/f4543c1e-ac24-4431-bbc7-65f950392636)

  * Фильтр. Если групп активных фильтров несколько, то появляется кнопка "Сбросить все", при нажатии на которую, все аткивные параметры выборки становятся неактивными.
  * Даты. При нажатии на кнопку с иконкой календаря и датой, появляется модальное окно
  * 
    ![image](https://github.com/user-attachments/assets/4fcccaa6-5db9-4126-a5d6-60a9b269d798)

  * Даты. При наведении на кнопки "Сегодня", "Вчера", "Эта неделя", "Прошлая неделя", "Этот месяц", "Прошлый месяц", "Последний 31 день", даты на календаре, попадающие в соответствующий диапазон, меняют свой цвет на более темный.
  * Даты. Сегдняшняя дата имеет нижнее подчеркивание. Крайние даты диапазона имеют синий цвет, а даты внутри диапазона имеют светло-серый цвет.
  * Даты. При нажатии на кнопки "Сегодня", "Вчера", "Эта неделя", "Прошлая неделя", "Этот месяц", "Прошлый месяц", "Последний 31 день", даты в полях над календарем, меняются на крайние, попадающие в выбранный диапазон. Даты на внутри календаря меняют свой стиль.
  * Даты. При нажатии на поле с датой, возможно ручное редактироание даты.
  * Даты. Если введенная дата (начала или конца диапазона) не входит в диапазон от Даты регистрации до Сегодняшней даты, поле меняет цвет на красный.
  * Даты. Если дата конца диапазона находится до даты начала диапазона, то они меняются местами.
  * Даты. При окончании ввода валидной даты в поле, значение внутри поля меняется на введенное и отображение дат внутри календаря менятеся на соответствующее новому диапазону.
  * Даты. При окончании ввода невалидной даты в поле, значение внутри поля меняется на Дату регистрации, если это дата начала, или на Сегодняшнюю дату, если это дата конца.
  * Даты. При нажатии на кнопку "Отменить", модальное окно закрывается без сохранения данных.
  * Даты. При нажатии на место вне модального окна, модальное окно закрывается без сохранения данных.
  * Даты. При нажатии на кнопку "Сохранить", новый временной диапазон сохранятся , становится активным и модальное окно заркывается.
  * Поиск. При вводе запроса в поле "Поиск", в списке изменений отображаются только те пункты, которые содержат тело запроса.
  

# Сайты
https://ads.vk.com/hq/pixels

- При нажатии на кнопку "Добавить пиксель" появляется модальное окно
  ![image](https://github.com/user-attachments/assets/35df2156-b153-4658-9429-22795f594dc3)

 * При выборе "Домен сайта" появляется текстовое поле "Домен сайта"
 * при нажатии на кнопку "Добавить пиксель", происходит добавление пекселя с введенным адресом.
 * * При вводе testforqa.ru происходит успешное добавление пикселя
     ![image](https://github.com/user-attachments/assets/74a2c175-50f5-46e2-b377-f3c8e65f677c)
 * * При вводе кириллицы тестдлякуа.ру происходит успешное добавление пикселя
   * При вводе цифр 1234567890.ru происходит успешное добавление пикселя
   * При вводе комбинированного адреса test4qa.privet происходит успешное добавление пикселя
 - На странице отображаются добавленные пиксели
 - При нажатии на адрес пикселя, в новой вкладке открывается указанный адрес
 - При нажатии на "Настройка" в графе пикселя, происходит переадресация на адрес "https://ads.vk.com/hq/pixels/*no_pixel*/events", где *no_pixel* это идентификатор выбранного пикселя (Например 3581887).
